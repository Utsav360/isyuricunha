Formação acadêmica

Universidado Estácio de Sá

Banco de dados• Conclusão no primeiro semestre de 2023

Com duração mínima de 2 anos e meio (5 períodos), o curso de Banco de Dados começa com a apresentação das matérias básicas da área de TI, como Organização de Computadores, Introdução à Programação, Fundamentos de Redes de Computadores e Cenários de TI. Durante o curso, são apresentadas as matérias que vão desde a Administração de SGBD (Sistema de gerenciamento de banco de dados) até a Análise de Dados. Nos preparando para exercer a função de DBA (Administrador de banco de dados). No último período, um Projeto Final que propõe solução de banco de dados para uma empresa encerra o curso. Diversas empresas utilizam sistemas de Bancos de Dados para organizar suas tarefas, clientes, funcionários, produtos, entre outros.
